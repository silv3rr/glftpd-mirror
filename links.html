glFTPd :: We make files transfer!

This is a list of glFTPd and glFTPd related sites. We thank all maintainers, scripters and programmers for their efforts.

Official glFTPd links:

  • archive.glftpd.io :: Your stop for a huge scripts archive and many many glFTPd releases.
  • bugs.glftpd.io :: Your stop for all glFTPd bug reports and feature requests.
  • github.com/glftpd :: Some support projects related to glftpd.

Other cool links:

  • www.glftpd.at :: Dvibe's very actively maintained site with all the scripts you could ever want.
  • pzs-ng :: Project Zipscript - Next Generation.
  • Chimera Coding :: Scripts by IP, dn, jehsom, all there!
  • FRSoldier :: FRSoldier's scripts.
  • Jehsom :: Jehsom used to be one of the most active bash gods out there, now's he idle, but left us a great script collection that are still widely used.
  • MaSa :: Masa has some nice tools to administer your glftpd site from a Windows desktop.
  • Turranius :: Turran as a great website where several others among him release their scripts.
  • Apocal :: Nice page, lot's of perl and bash scripts over there.
  • pnG :: Misc. glFTPd and shell scripts.
  • xferlogDB :: A PHP/MySQL tool for analyzing xferlog from glFTPd.
  • glFTPdWatcher :: GlFTPdWatcher is a small FREEWARE tool to help you handle SITE commands.
  • Zio :: Zio's glFTPd page with lots of scripts and links.
  • SoD :: SoD's home, with pre info, video_info, dated_dir_creator and more!
  • Multi-BNC :: Home of Multi-BNC.
  • glFTPd Site Manager :: gLFTPd Site Manager is a admin tool specially made for glFTPd server management.
  • SceneDivision :: SceneDivision is a place where you find scripts made by Darkstah / Nicon / Fairchild. Enjoy!
  • www.digital-tyranny.com/ :: www.digital-tyranny.com - The Online Answer to all your glFTPd scripting needs.
